## Basic commands

- `<C-b>c` - open a new window.
- `<C-b>|` - open a new pane, vertically split.
- `<C-b>-` - open a new pane, horizontally split.
- `<C-b>w` - select window.

## Starting new session on separate socket

(useful if you have terminals open with different themes)

`tmux -L socketname`
